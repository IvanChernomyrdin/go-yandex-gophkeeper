# GophKeeper

Клиент-серверное приложение для безопасного хранения паролей и секретных данных.

## Функции

- Регистрация и аутентификация пользователей
- Шифрованное хранение данных на клиенте
- Синхронизация с сервером
- Хранение разных типов секретов:
  - Логины и пароли
  - Текстовые заметки
  - Бинарные данные
  - Банковские карты
  - OTP-коды

## Установка

### Быстрый запуск:

go run launcher.go
